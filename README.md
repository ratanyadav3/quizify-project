# Quizify - Interactive Quiz Application

Quizify is an interactive quiz application built using HTML, CSS, and JavaScript. It allows users to test their knowledge in various categories with timed quizzes.

## Features

- **Multiple Categories**: Choose from General Knowledge, Computer Science, and Biology categories.
- **Timer**: Each question has a time limit to answer.
- **Score Tracking**: Track your score as you progress through the quiz.
- **Responsive Design**: Designed to work seamlessly on desktop and mobile devices.

## Technologies Used

- HTML5
- CSS3
- JavaScript
